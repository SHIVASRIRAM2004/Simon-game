## 🕹️ About the Game

Simon is a classic electronic memory game. The game lights up buttons in a random sequence, and the player must repeat the sequence in the correct order. Each round adds one more step — how far can your memory take you?

---

## ✨ Features

- 🎨 4 colored buttons — Green, Red, Yellow, Blue
- 🔊 Unique sound tone for each button (Web Audio API)
- 📈 Round counter and best score tracker
- ⚡ Speed increases as rounds progress
- 💀 Strict Mode — one wrong press ends the game
- ❌ Error flash animation with buzzer sound
- 📱 Fully responsive — works on desktop and mobile
- 🎰 Retro arcade aesthetic with scanline effect

---

## 🚀 How to Play

1. Click **START** to begin
2. Watch the colored buttons light up in a sequence
3. Click the buttons in the **same order**
4. Each round adds one more step to the sequence
5. Wrong press? In normal mode you retry — in **Strict Mode** it's game over!
